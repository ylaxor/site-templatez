# SaaS Site Template

This is an auto-provisioned website template with content management capabilities.

## Features

- Home page with editable content
- Board page for content management
- Firebase integration for data persistence
- Live content editing

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view your site.

## Content Management

Navigate to `/board` to manage your site content. Login and edit content directly.
